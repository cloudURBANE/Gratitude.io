# TipLink (UI-first design demo)

A polished Next.js + Tailwind CSS scaffold for a **universal tip + review** landing page. This is **design-only**: no backend, auth, or real payment logic yet — just the UX so you can demo, iterate, and sell.

## Quick start

```bash
# Node 18.17+
npm install
npm run dev
# open http://localhost:3000
```

## Routes

- `/` — simple marketing hero
- `/styleguide` — design tokens + components
- `/u/[handle]` — QR landing (try `/u/jordan`)
- `/u/[handle]/edit` — demo editor (no persistence)
- `/success` — thanks page with confetti

## Notes

- **All icons are inline SVG**, no external CDN.
- **Accessible**: focusable controls, ESC/Tab trapping in modal, visible rings.
- **Mobile-first**: designed to look great at 390×844 and up.
- **Design tokens** live in `tailwind.config.js` and `globals.css`.
